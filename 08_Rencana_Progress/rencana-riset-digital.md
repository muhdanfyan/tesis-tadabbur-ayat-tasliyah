# Rencana Riset Digital: Integrasi Tadabbur Al-Quran & Psikologi

> **Platform:** Wajar.ai (Wadah Analisis Journaling & Assessment Riset)
> **Fokus:** QS Ad-Dhuha & QS Al-Insyirah sebagai Intervensi Psikologis
> **Metode:** Mixed Methods (Eksperimen + Analisis Tematik)

---

## A. ROADMAP PENELITIAN (4 Fase — 6 Bulan)

### Fase 1: Persiapan (Bulan 1)

**Aktivitas:**
1. **Studi Literatur Intensif**
   - Finalisasi kajian tafsir Ibn Katsir, Al-Razi, As-Sa'di, Hamka
   - Review lengkap psikologi: attachment, optimism, resilience, gratitude, grit
   - Mapping titik temu ayat ↔ teori (15 item)

2. **Penyusunan Modul Intervensi**
   - Modul tadabbur QS Ad-Dhuha (11 ayat × 3 sesi)
   - Modul tadabbur QS Al-Insyirah (8 ayat × 2 sesi)
   - Panduan journaling 3 kolom (ayat, makna, refleksi)

3. **Penyusunan Instrumen Penelitian**
   Adaptasi dan validasi instrumen psikologi:
   - RASA (Religious Attachment Scale) — adaptasi dari ECR-R
   - LOT-R (Life Orientation Test) — optimisme
   - GSES (General Self-Efficacy Scale) — efikasi diri
   - GQ-6 (Gratitude Questionnaire) — syukur
   - GRIT-S (Short Grit Scale) — kegigihan
   - DASS-42 (Depression Anxiety Stress Scale) — kesehatan mental
   - SPWB (Scala Psychological Well-Being) — kesejahteraan psikologis
   - Skala Tadabbur (dikembangkan sendiri) — validasi isi oleh 3 ahli

4. **Pembangunan Platform Wajar.ai**
   - Setup landing page rekrutmen
   - Integrasi WhatsApp Bot
   - Template e-form informed consent
   - Template assessment digital
   - Modul pembelajaran tadabbur (video, audio, PDF)
   - Journaling form 3 kolom digital
   - Dashboard monitoring sederhana
   - Database partisipan

5. **Ethical Clearance**
   - Pengajuan ke komite etik (fakultas/universitas)
   - Penyiapan informed consent form

**Persiapan:**
- Izin riset (universitas/kampus/fakultas)
- Kerjasama dengan program studi psikologi Islam/umum
- Pilot uji coba instrumen (30 partisipan)
- Revisi modul dan instrumen

**Estimasi Waktu:** 4 minggu (Minggu 1-4)

**Pihak Terlibat:**
- Peneliti utama (Mahasiswa S2/S3)
- Dosen pembimbing (2 orang)
- Ahli psikologi (validasi instrumen)
- Ahli tafsir (validasi modul tadabbur)
- Developer web (Wajar.ai)

**Output:**
- ✓ Modul intervensi tadabbur final
- ✓ Instrumen tervalidasi
- ✓ Platform Wajar.ai versi beta
- ✓ Ethical clearance
- ✓ Dokumen persiapan (SOP, timeline, budget)

### Fase 3: Analisis (Bulan 4-5)

**Aktivitas:**

1. **Data Cleaning & Coding (Minggu 11-12)**
   - Ekspor data dari database Wajar.ai
   - Pembersihan data (missing values, outliers)
   - Coding data kualitatif (journaling, wawancara)
   - Inter-rater reliability untuk coding

2. **Analisis Kuantitatif (Minggu 13-14)**
   **Menggunakan SPSS / R (script otomatis):**
   
   **Analisis Deskriptif:**
   - Mean, SD, distribusi frekuensi tiap variabel
   - Profil demografis partisipan
   
   **Uji Asumsi:**
   - Normalitas (Shapiro-Wilk)
   - Homogenitas (Levene's Test)
   
   **Uji Hipotesis:**
   - Independent t-test / Mann-Whitney (kelompok eksperimen vs kontrol)
   - Paired t-test / Wilcoxon (pre-test vs post-test per kelompok)
   - ANCOVA (mengontrol variabel kovariat)
   - Effect size (Cohen's d)
   
   **Analisis Tambahan:**
   - Korelasi Pearson (antar variabel)
   - Regresi (prediktor perubahan well-being)
   - Analisis mediasi (jika model teoritis mendukung)

3. **Analisis Kualitatif (Minggu 14-15)**
   **Thematic Analysis (Braun & Clarke, 2006):**
   - Familiarisasi data (membaca journaling & transkrip)
   - Initial coding (open coding)
   - Searching for themes
   - Reviewing themes
   - Defining and naming themes
   - Writing up

   **Fokus Tematik:**
   - Perubahan perspektif partisipan
   - Pengalaman tadabbur
   - Kaitan dengan psikologi positif

4. **Triangulasi & Interpretasi (Minggu 15-16)**
   - Menghubungkan temuan kuantitatif dan kualitatif
   - Interpretasi dalam kerangka tafsir (Ad-Dhuha & Al-Insyirah)
   - Interpretasi dalam kerangka psikologi
   - Diskusi dengan pembimbing

**Persiapan:**
- Script SPSS/R untuk analisis
- Panduan coding tematik
- Matriks triangulasi

**Estimasi Waktu:** 8 minggu (Minggu 11-18)

**Pihak Terlibat:**
- Peneliti utama
- Dosen pembimbing (review hasil analisis)
- Asisten riset (bantu coding kualitatif)
- Ahli statistik (jika perlu konsultasi)

**Output:**
- ✓ Tabel hasil statistik
- ✓ Tema-tema kualitatif
- ✓ Hasil triangulasi
- ✓ Draft temuan untuk diskusi

### 7. Analisis → SPSS/R Otomatis

**Deskripsi:**
Analisis data statistik dengan script otomatis.

**Digitalisasi:**
**R Script Otomatis:**
```r
# Analisis otomatis untuk tesis tadabbur
library(dplyr)
library(psych)
library(effsize)

# Load data dari database
data <- read.csv('output_prepost.csv')

# Uji normalitas
shapiro.test(data$post_optimism)

# Uji t (eksperimen vs kontrol)
t.test(post_wellbeing ~ group, data = data)

# Effect size
cohens_d(post_wellbeing ~ group, data = data)

# ANCOVA
ancova_model <- aov(post_wellbeing ~ pre_wellbeing + group, data = data)
summary(ancova_model)
```

**Fitur:**
- Script R template untuk analisis standar

**Tools:**

## C. KONSEP BISNIS WAJAR.AI

### Identitas Platform

| Aspek | Detail |

### Value Proposition

**Masalah:**
1. Riset psikologi Islam masih manual — kertas, Excel, antrian panjang
2. Belum ada platform khusus untuk intervensi tadabbur
3. Data riset tercecer di banyak tools (Google Form, Drive, WA)
4. Analisis data butuh waktu lama dan skill teknis

**Solusi:**
Wajar.ai menyediakan **satu platform all-in-one** untuk seluruh siklus riset psikologi Islam berbasis tadabbur:

| Tahap | Masalah | Solusi Wajar.ai |
|-------|---------|-----------------|
| Rekrutmen | Nyebar poster manual, data campur aduk | Landing page + WA Bot otomatis |
| Informed Consent | TTD kertas, arsipkan di laci | E-form + database cloud |
| Assessment | Print kuesioner, input manual ke Excel | Survey digital + auto-scoring |
| Intervensi | Ngajar tatap muka, repot jadwal | Modul online (video, audio, PDF) |
| Journaling | Buku catatan, susah dianalisis | Form 3 kolom digital + export |
| Monitoring | Mahasiswa ngechat "Bu, sudah?" | Dashboard progress real-time |
| Analisis | SPSS manual, copy-paste, rawan error | Script otomatis + visualisasi |
| Drafting | Referensi berserakan, format manual | Google Docs + Zotero integrasi |
